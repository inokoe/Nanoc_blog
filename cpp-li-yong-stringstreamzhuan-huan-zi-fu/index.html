Cpp 利用stringstream转换字符

二月 5, 2022 by Nanoc

stringstream

stringstream是包含在中的，其中定义了三个类：istringstream、ostringstream 和 stringstream，分别用来进行流的输入、输出和输入输出操作。

用法

#include <bits/stdc++.h>
using namespace std;

int main()
{
    stringstream sts;
    string x = "10"; 
    int z = 100;
    string y = "";

    //string 给 string
    sts<<x;
    y = sts.str();
    cout<<y<<endl;
    
    //清除缓冲区
    /*
    在多次数据类型转换的场景下，必须使用 clear() 方法清空 stringstream，不使用 clear()方法或使用 str("") 方法，都不能得到数据类型转换的正确结果。
    */
    sts.str("");
    sts.clear();

    //int 给 string
    sts<<z;
    x = sts.str();
    cout<<x<<endl;

    return 0;
}
