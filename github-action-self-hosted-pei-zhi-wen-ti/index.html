Github Action Self hosted 配置问题

二月 26, 2022 by Nanoc

Must not run with sudo

您可以通过创建用户来解决或者如下:

修改 ./config.sh 内容

vi ./config.sh

删除这部分

if [ $user_id -eq 0 -a -z "$AGENT_ALLOW_RUNASROOT" ]; then
    echo "Must not run with sudo"
    exit 1
fi

或者在命令前部添加：

RUNNER_ALLOW_RUNASROOT="1"
