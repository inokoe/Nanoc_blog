Python 文件读写操作

二月 5, 2022 by Nanoc

读写模式

值描述
r读模式
w写模式
a追加模式
b二进制模式（可追加到其他模式中）
+读/写模式（可追加到其他模式中
# 追加读写模式 自动创建文件
f = open("demo.txt", 'a+', encoding="utf-8")
f.write(soup.text)
f.close()

# 覆盖 自动创建文件
f = open("C:/Users/Inokoe/Desktop/demo.txt", 'w', encoding="utf-8")
f.write('653') 
f.close()

f = open("C:/Users/Inokoe/Desktop/demo.txt", 'w+', encoding="utf-8")
f.write('653') 
# 定位到开头可以正常读取
f.seek(0)
print(f.read())
f.close()

# 只读
f = open("C:/Users/Inokoe/Desktop/demo.txt", 'r', encoding="utf-8")
f.write('653') 
f.close()
